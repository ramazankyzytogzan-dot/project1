# AWS EKS ортасындағы үш деңгейлі DevSecOps жобасы

Бұл жоба аясында **AWS EKS (Elastic Kubernetes Service)** инфрақұрылымында ReactJS, NodeJS және MongoDB негізіндегі үш деңгейлі веб-қосымшаны автоматтандырылған түрде орналастыру процесін жасап шықтым. Жобаның басты ерекшелігі — **Gemini AI** нейрожелісі мен **n8n** платформасын қолдана отырып, интеллектуалды мониторинг жүйесін біріктіру.

---

## Пайдаланылған технологиялар

### Инфрақұрылым (IaC)
* **Terraform**: Инфрақұрылымды код арқылы құру.
* **AWS CLI & EKS**: AWS ресурстарын басқару және Kubernetes кластері.

### CI/CD және автоматтандыру
* **Jenkins**: Пайплайндарды автоматтандыру.
* **SonarQube**: Код сапасын тексеру.
* **Docker & AWS ECR**: Контейнерлермен жұмыс.
* **ArgoCD**: GitOps арқылы қосымшаны автоматты жаңарту.

### Мониторинг және AI
* **Prometheus & Grafana**: Мәліметтер жинау және бақылау.
* **n8n + Google Gemini AI**: Қателерді талдайтын жасанды интеллект жүйесі.

---

## Интеллектуалды мониторинг қалай жұмыс істейді?

Жобада мониторинг деректерін талдаудың келесі алгоритмін іске асырдым:

1. **Prometheus** кластердегі ақауларды немесе шекті мәндерден ауытқуды анықтайды.
2. **n8n** платформасы HTTP Webhook арқылы деректерді қабылдап, оны **Google Gemini AI** моделіне жібереді.
3. Нейрожелі техникалық логтарды талдап, ақаудың себебі мен оны шешу жолдарын (қазақ/орыс тілдерінде) дайындайды.
4. Қорытынды есеп **Telegram** арқылы жедел хабарлама түрінде келеді.

---

## Каталогтардың сипаттамасы

| Папка атауы | Атқаратын қызметі |
| :--- | :--- |
| **Application-Code** | Веб-қосымшаның бастапқы коды (Frontend/Backend). |
| **Jenkins-Pipeline-Code** | CI/CD процесін сипаттайтын кодтар. |
| **Jenkins-Server-TF** | Jenkins серверін құруға арналған Terraform файлдары. |
| **Kubernetes-Manifests-file** | Kubernetes-ке арналған YAML манифестері. |

---

## Жұмыс барысы
1. `Jenkins-Server-TF` арқылы қажетті серверлерді баптадым.
2. Jenkins-те CI пайплайндарын реттеп, Docker бейнелерін ECR-ға жібердім.
3. ArgoCD көмегімен Kubernetes кластеріне қосымшаны орналастырдым.
4. Мониторинг пен Gemini AI интеграциясын қостым.
