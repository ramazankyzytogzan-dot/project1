AWS EKS ортасында ArgoCD, Prometheus және Gemini AI негізіндегі үш деңгейлі қосымшаны орналастыру (DevSecOps)
Бұл жоба AWS EKS (Elastic Kubernetes Service) инфрақұрылымында ReactJS, NodeJS және MongoDB негізіндегі үш деңгейлі веб-қосымшаны автоматтандырылған түрде орналастыру процесін қамтиды. Жобаның басты ерекшелігі — Gemini AI нейрожелісі мен n8n платформасы арқылы интеллектуалды мониторинг жүйесін енгізу.

Жобаның құрылымы мен технологиялық стекі
Инфрақұрылым (IaC): Terraform, AWS CLI, AWS EKS.

CI/CD және автоматтандыру: Jenkins (Pipeline-as-code), SonarQube (статикалық код талдауы), Docker, AWS ECR.

GitOps: ArgoCD (Continuous Delivery).

Мониторинг және жасанды интеллект: Prometheus, Grafana, n8n, Google Gemini AI.

Қосымша: ReactJS (Frontend), NodeJS (Backend), MongoDB (Database).

Интеллектуалды мониторинг жүйесі (AI Integration)
Жоба аясында мониторинг деректерін талдаудың келесі алгоритмі іске асырылды:

Prometheus жүйесі кластердегі ақауларды немесе шекті мәндерден ауытқуды анықтайды.

n8n платформасы HTTP Webhook арқылы деректерді қабылдап, оны Google Gemini AI моделіне жолдайды.

Нейрожелі техникалық логтарды талдап, ақаудың себебі мен оны шешу жолдарын (орыс/қазақ/ағылшын тілдерінде) қысқаша тұжырымдайды.

Қорытынды есеп Telegram мессенджеріне жедел хабарлама түрінде жіберіледі.

Каталогтардың сипаттамасы
Application-Code: Веб-қосымшаның бастапқы коды (Frontend және Backend).

Jenkins-Pipeline-Code: Кодты автоматты түрде жинауға, тестілеуге және Docker бейнелерін жасауға арналған сценарийлер.

Jenkins-Server-TF: AWS ортасында Jenkins серверін баптауға арналған Terraform кодтары.

Kubernetes-Manifests-Files: Қосымшаны EKS кластерінде жариялауға арналған Kubernetes манифесттері.

Негізгі орындалған жұмыс кезеңдері
AWS платформасында IAM рөлдерін және Terraform көмегімен VPC/EKS инфрақұрылымын баптау.

Jenkins платформасында SonarQube және Docker-мен интеграцияланған толық CI/CD конвейерін құру.

ArgoCD көмегімен GitOps принциптеріне сәйкес үздіксіз жариялауды (CD) ретке келтіру.

n8n және Gemini AI негізіндегі "ақылды" хабарландыру жүйесін қосу арқылы жүйенің бақылану мүмкіндігін арттыру.

Қорытынды
Бұл жоба заманауи DevSecOps тәжірибелерін ғана емес, сонымен қатар бұлттық инфрақұрылымды басқаруда жасанды интеллект мүмкіндіктерін тиімді пайдалану жолдарын көрсетеді.
